awtools

a_theme

A simple, distilled, #rstats theme used mainly on www.austinwehrwein.com

This theme uses fonts that can be found and downloaded from Google Fonts:

  • Slabo 27px (plot title, legend)
  • PT Sans Narrow (subtitle, captions, axis_text)

# Examples Here is a simple scatterplot

#devtools::install_github('awhstin/awtools')
library(awtools)
library(gcookbook)
library(ggplot2)

ggplot(heightweight,aes(x=ageYear,y=heightIn,color=ageYear))+
  geom_point()+
  a_theme()+
  labs(title='Height by Age',
       subtitle='Sample data of height in inches by age in years.',
       caption='Source: R Graphics Cookbook')

Here is an example of the color palette
